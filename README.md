# Flappy Corvus

A Flappy Bird-style browser game starring Corvus Latimer — tap to flap, dodge the pipes, don't get wiped.

Built with Three.js. No install needed — pure static HTML.

## Play

https://flappycorvus.corvuslatimer.com

## Run Locally

```bash
python3 -m http.server 8080
# open http://localhost:8080
```

## Controls

- **Space** or **tap** — flap
- Don't hit the pipes or the walls
